# OpenClaw v2026.4.9 Secure Integration Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Integrate OpenClaw v2026.4.9 as IliaGPT's private backend execution engine with full tenant isolation, gateway security, per-turn usage tracking, quota enforcement, and RBAC — keeping IliaGPT as the authoritative layer for auth, billing, and governance.

**Architecture:** OpenClaw remains the orchestration/runtime kernel (tools, skills, subagents, streaming) but is fully encapsulated behind IliaGPT's Express backend. The WebSocket gateway gets JWT-based auth so only authenticated server-side code can connect. Sessions are scoped by `tenant:user:chat` composite keys stored in PostgreSQL. A `usage_ledger` table records every LLM turn with tokens, cost, model, and user. Quota enforcement runs pre-execution and returns `TOKEN_QUOTA_EXCEEDED` when limits are hit.

**Tech Stack:** Express.js, PostgreSQL (Drizzle ORM), prom-client (Prometheus), ioredis, jose (JWT), WebSocket (ws), existing LLM gateway.

// server/openclaw/gateway/auth.ts
import * as jose from "jose";
import { env } from "../../config/env";

const ALG = "HS256";

function getSecret(): Uint8Array {
  return new TextEncoder().encode(env.OPENCLAW_GATEWAY_SECRET);
}

export interface GatewayTokenPayload {
  sub: string;       // userId
  tid: string;       // tenantId (orgId)
  cid: string;       // chatId / conversationId
  role: string;      // admin | user | guest
  iat: number;
  exp: number;
}

/** Sign a gateway token (called by IliaGPT backend before opening WS). */
export async function signGatewayToken(payload: {
  userId: string;
  tenantId: string;
  chatId: string;
  role: string;
}): Promise<string> {
  return new jose.SignJWT({
    sub: payload.userId,
    tid: payload.tenantId,
    cid: payload.chatId,
    role: payload.role,
  })
    .setProtectedHeader({ alg: ALG })
    .setIssuedAt()
    .setExpirationTime("1h")
    .sign(getSecret());
}

/** Verify a gateway token from WS handshake query param. */
export async function verifyGatewayToken(token: string): Promise<GatewayTokenPayload> {
  const { payload } = await jose.jwtVerify(token, getSecret(), {
    algorithms: [ALG],
  });
  return payload as unknown as GatewayTokenPayload;
}
```

// server/openclaw/gateway/tenantContext.ts

export interface TenantContext {
  tenantId: string;
  userId: string;
  chatId: string;
  role: "admin" | "user" | "guest";
  /** Composite isolation key: tenant:user:chat */
  sessionScope: string;
}

export function buildTenantContext(params: {
  tenantId: string;
  userId: string;
  chatId: string;
  role: string;
}): TenantContext {
  return {
    tenantId: params.tenantId,
    userId: params.userId,
    chatId: params.chatId,
    role: (params.role === "admin" || params.role === "guest") ? params.role : "user",
    sessionScope: `${params.tenantId}:${params.userId}:${params.chatId}`,
  };
}
```

Replace the `wss.on('connection')` handler to verify the JWT token from the query string before accepting the connection. Reject with 4001 (Unauthorized) if missing or invalid.

Key changes:

2. Import `buildTenantContext` from `./tenantContext`

4. Verify token, build `TenantContext`, store on `ConnectedClient`
5. Pass `TenantContext` to `handleRpc` instead of bare `{ userId }`
6. On verification failure: `ws.close(4001, 'Unauthorized')`

```typescript
// In the connection handler:
const token = url.searchParams.get('token');
if (!token) {
  ws.close(4001, 'Missing gateway token');
  return;
}

let tenantCtx: TenantContext;
try {
  const payload = await verifyGatewayToken(token);
  tenantCtx = buildTenantContext({
    tenantId: payload.tid,
    userId: payload.sub,
    chatId: payload.cid,
    role: payload.role,
  });
} catch {
  ws.close(4001, 'Invalid gateway token');
  return;
}
```

## Task 9: Create quota gate

**Files:**
- Create: `server/services/openclawQuotaGate.ts`

- [ ] **Step 1: Write the quota gate**

```typescript
// server/services/openclawQuotaGate.ts
import { getUserDailyUsage } from "./usageLedgerService";
import { db } from "../db";
import { users } from "@shared/schema";
import { eq } from "drizzle-orm";

export interface QuotaCheckResult {
  allowed: boolean;
  code?: "TOKEN_QUOTA_EXCEEDED" | "DAILY_LIMIT_EXCEEDED";
  message?: string;
  remaining?: number;
}

const DAILY_TOKEN_LIMITS: Record<string, number> = {
  free: 100_000,
  go: 1_000_000,
  plus: 5_000_000,
  pro: -1,       // unlimited
  admin: -1,
  enterprise: -1,
};

export async function checkQuota(userId: string): Promise<QuotaCheckResult> {
  const [user] = await db.select({ plan: users.plan, role: users.role })
    .from(users).where(eq(users.id, userId)).limit(1);

  if (!user) return { allowed: false, code: "TOKEN_QUOTA_EXCEEDED", message: "User not found" };

  const plan = user.plan || "free";
  const role = String(user.role || "").toLowerCase();
  if (role === "admin" || role === "superadmin") return { allowed: true };

  const limit = DAILY_TOKEN_LIMITS[plan] ?? DAILY_TOKEN_LIMITS.free;
  if (limit === -1) return { allowed: true };

  const usage = await getUserDailyUsage(userId);
  const remaining = limit - usage.totalTokens;

  if (remaining <= 0) {
    return {
      allowed: false,
      code: "TOKEN_QUOTA_EXCEEDED",
      message: `Has alcanzado tu límite diario de ${limit.toLocaleString()} tokens (plan ${plan}). Recarga o mejora tu plan.`,
      remaining: 0,
    };
  }

  return { allowed: true, remaining };
}
```
